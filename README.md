## Visão Geral

O **Atlas Text Editor** foi projetado para ser uma ferramenta de edição de texto rápida e confiável. Seja para fazer anotações rápidas, visualizar logs ou editar arquivos grandes, o Atlas oferece uma interface simples sem consumir todo os recursos do seu computador.

## Instalação

O Atlas Text Editor está disponível para **Windows** e **Linux**. Para instalar, basta ir até a aba [Releases](https://github.com/hptsilva/atlas-text-editor/releases) deste repositório e baixar o arquivo compatível com o seu sistema.

### Windows
1. Baixe o arquivo **`Atlas-Setup-x64.exe`** na página de Releases.
2. Dê um duplo clique no instalador e siga as instruções na tela.
3. O Atlas será adicionado na Área de Trabalho!

### Linux

Temos pacotes pré-compilados para as principais distribuições:

**Para Ubuntu, Debian, Mint e derivados (.deb):**
1. Baixe o arquivo **`atlas-text-editor_1.1.0_amd64.deb`**.
2. Instale dando um duplo clique (se tiver uma loja de aplicativos) ou pelo terminal:
   ```bash
   sudo dpkg -i atlas-text-editor_1.1.0_amd64.deb
   ```

**Para Fedora, Red Hat, CentOS e derivados (.rpm):**
1. Baixe o arquivo **`atlas-text-editor-1.1.0-1.x86_64.rpm`**.
2. Instale dando um duplo clique (se tiver um loja de aplicativos) ou pelo terminal:
   ```bash
   sudo rpm -i atlas-text-editor-1.1.0-1.x86_64.rpm
   ```
